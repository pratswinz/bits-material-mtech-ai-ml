Q1 — Electricity Bill Regression [5M]

\(Bill = 120 + 7.5X_1 + 45X_2 - 4X_3 + 18X_4\) · kWh, AC count, temp °C, family size

(a)(i) Interpret coefficients

\(w_0=120\): base bill $120 · \(w_1=7.5\): +$7.50 per kWh · \(w_2=45\): +$45 per AC · \(w_3=-4\): −$4 per °C · \(w_4=18\): +$18 per person

(a)(ii) kWh → Wh

1 kWh = 1000 Wh → coefficient becomes \(7.5/1000 = 0.0075\) per Wh. Prediction unchanged if input scaled consistently.

(b) \(\partial J/\partial\theta_3 = -12\)

Gradient descent
\(\theta_j \leftarrow \theta_j - \alpha\frac{\partial J}{\partial\theta_j}\)
Negative gradient → increasing \(\theta_3\) lowers cost → GD increases weight: \(\theta_3 \leftarrow \theta_3 + 12\alpha\).

(c) Strongest feature?

Largest |coef| = AC count (45). Caveat: compare on same scale — raw magnitude misleading before normalization.

Q2 — Decision Tree: Loan Approval

10 applicants: 6 Approved, 4 Denied

\(p_{Yes}=0.6,\ p_{No}=0.4\)
\(H = -0.6\log_2(0.6) - 0.4\log_2(0.4) = 0.442 + 0.529\)
Entropy ≈ 0.971 bits

Bonus — IG on "Existing Loan = No"

No loan branch: 5 Yes, 0 No → H=0 · Has loan: 1 Yes, 4 No → H≈0.722
Weighted H = 0.361 · IG = 0.971 − 0.361 = 0.610

TYPE 5 — Bias-Variance

Polynomial deg-2 vs deg-15

A (deg 2): train≈test → high bias · B (deg 15): train≈0, test high → high variance
Don't use higher degree — use regularization, CV, more data

4 models RMSE — A best, B overfit, C underfit, D decent

R² = 1 − SS_res/SS_tot · B diverges from memorizing noise

"Doubled data but val error still high"

High bias — more data won't help. Add features / complexity / reduce λ.
