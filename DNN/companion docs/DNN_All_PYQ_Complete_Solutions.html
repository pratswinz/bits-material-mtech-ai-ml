Mid Q1 — MLP Design [6 marks]HIGH PRIORITY

Design minimum hidden layers/nodes for non-linear boundary. Step: +1 if sum>=threshold else -1.
Understand
Single perceptron = ONE linear boundary.
Concentric circles NOT linearly separable (XOR principle in 2D).
One line cannot separate inner +1 from outer -1 ring.
Design
Input(2) -> Hidden(2) -> Output(1)
H1: inner circle detector
H2: outer circle detector
Output: combine for +1 inner, -1 outer
Minimum: 1 hidden layer, 2 nodes
Exam draw
Draw diagram with THRESHOLD inside each node.
Show weights on connections.
Examiners want WHY hidden layer is needed.
